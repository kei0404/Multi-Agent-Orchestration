# README Generator

Pythonプロジェクト向けの日本語README.md自動生成スキル。

## ワークフロー

### 1. プロジェクト分析

以下のファイル/ディレクトリを調査してプロジェクトを理解する：

```
必須チェック:
- pyproject.toml または setup.py（プロジェクト名、依存関係）
- requirements.txt（依存関係）
- メインモジュール（src/ または プロジェクト名ディレクトリ）
- tests/（テスト構造）

任意チェック:
- .env.example（環境変数）
- Makefile または Taskfile（コマンド）
- docker-compose.yml（Docker設定）
```

### 2. README構成

以下のセクションを含むREADME.mdを生成する：

```markdown
# プロジェクト名

## 概要
[プロジェクトの目的と主要機能を2-3文で説明]

## 機能
- 機能1の説明
- 機能2の説明
- ...

## インストール

### 前提条件
- Python 3.x 以上
- [その他の前提条件]

### セットアップ
[インストール手順をコマンド付きで記載]

## 使い方

### 基本的な使用例
[コード例を含める]

### API リファレンス
[主要な関数/クラスの説明]

## ディレクトリ構造
[プロジェクト構造をツリー形式で表示]

## 開発

### テストの実行
[テストコマンド]

### リンター
[リンターコマンド]
```

### 3. 生成ルール

- **言語**: 日本語で記述
- **コード例**: 実際のプロジェクトコードから抜粋または作成
- **簡潔さ**: 各セクションは必要最小限の情報に絞る
- **正確性**: pyproject.toml等から正確な情報を取得

## テンプレート

テンプレートファイル: [assets/readme_template.md](assets/readme_template.md)

基本テンプレートをベースに、プロジェクト固有の情報で置換する。

## 出力

生成したREADME.mdはプロジェクトルートに配置する。既存のREADME.mdがある場合は上書き前に確認を求める。
